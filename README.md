# AI Workflow Agent Console

A staged AI workflow prototype for client operations. The system demonstrates discovery, workflow planning, runtime exception triage, Airtable audit records, Slack alerts, and human-review routing.

## Prototype
https://melior-ai-triage-console-xp23.vercel.app/

## What it demonstrates
* Discovery Agent for messy client notes
* Workflow Architect Agent for sprint-ready MVP workflow plans
* Runtime Exception Triage for staged support tickets
* n8n workflow orchestration
* Airtable audit records
* Slack handoff
* Human review queue
* 10-case staged evaluation

## Stack
Next.js, TypeScript, n8n, Airtable, Slack, JavaScript normalization layer, Claude / OpenAI-compatible model endpoint

## Notes
This is a staged MVP prototype, not a production deployment. The goal is to demonstrate workflow architecture, handoff logic, human review, auditability, and staged reliability testing.
